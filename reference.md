# Linear Command Reference

Use this reference to understand the commands and parameters available in Linear for effective product development. Each command is categorized for easy navigation.

## AI Agent Commands

| Parameter/Command             | Type         | Description                                           | Example                          |
|-------------------------------|--------------|-------------------------------------------------------|----------------------------------|
| `create_agent`                | Function     | Creates a new AI agent to assist with tasks.         | `create_agent("Cursor")`        |
| `deploy_agent`                | Function     | Deploys an AI agent to work alongside your team.     | `deploy_agent("Cursor")`        |
| `delegate_task`               | Function     | Assigns a task to an AI agent for completion.        | `delegate_task("Fix bug", "Cursor")` |
| `train_agent`                 | Function     | Trains an AI agent on specific tasks or datasets.    | `train_agent("Cursor", "bug fixes")` |

## Issue Management Commands

| Parameter/Command             | Type         | Description                                           | Example                          |
|-------------------------------|--------------|-------------------------------------------------------|----------------------------------|
| `create_issue`                | Function     | Creates a new issue in the project.                  | `create_issue("UI Bug")`        |
| `assign_issue`                | Function     | Assigns an issue to a team member or agent.          | `assign_issue("UI Bug", "Karri")` |
| `update_issue_status`         | Function     | Updates the status of an existing issue.              | `update_issue_status("UI Bug", "In Progress")` |
| `close_issue`                 | Function     | Closes an issue once resolved.                        | `close_issue("UI Bug")`         |

## Project Management Commands

| Parameter/Command             | Type         | Description                                           | Example                          |
|-------------------------------|--------------|-------------------------------------------------------|----------------------------------|
| `create_project`              | Function     | Creates a new project for development.                | `create_project("New Feature")` |
| `add_team_member`             | Function     | Adds a team member to a project.                      | `add_team_member("New Feature", "Jori")` |
| `set_project_priority`        | Function     | Sets the priority level for a project.                | `set_project_priority("New Feature", "High")` |
| `archive_project`             | Function     | Archives a completed project.                          | `archive_project("Old Feature")` |

## Performance Commands

| Parameter/Command             | Type         | Description                                           | Example                          |
|-------------------------------|--------------|-------------------------------------------------------|----------------------------------|
| `optimize_performance`        | Function     | Optimizes the performance of the application.         | `optimize_performance()`         |
| `monitor_performance`         | Function     | Monitors the performance metrics of the application.  | `monitor_performance()`          |
| `generate_performance_report` | Function     | Generates a report on application performance.        | `generate_performance_report()`  |

## Collaboration Commands

| Parameter/Command             | Type         | Description                                           | Example                          |
|-------------------------------|--------------|-------------------------------------------------------|----------------------------------|
| `invite_collaborator`         | Function     | Invites a collaborator to join the project.          | `invite_collaborator("Cristina")` |
| `share_project`               | Function     | Shares the project with stakeholders.                 | `share_project("New Feature")`   |
| `comment_on_issue`            | Function     | Adds a comment to an existing issue.                  | `comment_on_issue("UI Bug", "Looks good!")` |

Utilize these commands to streamline your product development process with Linear, enhancing collaboration and efficiency in your team.
